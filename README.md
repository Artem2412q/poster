# АвтоПостер — премиальный сайт (v2)

Статический сайт: HTML/CSS/JS (без сервера).
- Корзина хранится в `localStorage`
- «Оформить в Telegram» открывает чат `@artem_myuu` и подставляет готовый текст заказа
- Поддержка тёмной/светлой темы (переключатель в шапке)
  - Dark: только чёрный/серый
  - Light: белый/серый/чёрный/бежевый
- Адаптивные анимации (с учётом `prefers-reduced-motion`)
- Адаптивная галерея примеров работ

## Запуск
Открой `index.html` в браузере  
или запусти локальный сервер:
`python -m http.server 8080`

## Где менять цены / размеры
`app.js` → массив `PRODUCTS[0].options`

## Где менять Telegram
`app.js` → `TG_USERNAME`

## Где менять тексты
`index.html`
